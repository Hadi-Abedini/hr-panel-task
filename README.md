## امکانات

- مشاهده درخواست‌ها بر اساس وضعیت
- جستجو و فیلتر پیشرفته
- مدیریت و بررسی درخواست‌ها
- بازگشت درخواست به مرحله قبل
- عملیات گروهی روی درخواست‌ها
- نمایش جزئیات کامل هر درخواست
- رابط کاربری RTL و فارسی
- طراحی کاملاً Responsive
